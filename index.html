Preguntas frecuentes

Lo que necesitás saber

¿Necesito internet para usarlo?

El sistema está diseñado para funcionar en una sola PC con base de datos local SQLite.

¿Imprime factura fiscal?

No. ComerCiando emite tickets internos no fiscales para impresoras térmicas comunes de 80 mm.

¿Puedo vender sin imprimir?

Sí. Podés finalizar una venta con o sin impresión de ticket.

¿Puedo respaldar la información?

Sí. Incluye backup manual de la base local y exportaciones CSV básicas.
