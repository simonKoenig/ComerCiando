Preguntas frecuentes

Lo que necesitás saber

¿Necesito internet para usarlo?

El sistema está diseñado para funcionar en una sola PC con base de datos local SQLite.

¿Imprime factura fiscal?

No. ComerCiando emite tickets internos no fiscales para impresoras térmicas comunes de 80 mm.

¿Puedo vender sin imprimir?

Sí. Podés finalizar una venta con o sin impresión de ticket.

¿Puedo respaldar la información?

Sí. Incluye backup manual de la base local y exportaciones CSV básicas.

¿Puedo importar los productos que ya tengo?

Sí. Podés preparar una planilla CSV desde Excel e importar productos y precios para empezar más rápido.

¿Puedo registrar clientes que compran fiado?

Sí. Podés finalizar una venta como fiado, registrar pagos parciales y consultar quiénes tienen saldo pendiente.

¿Puedo controlar lo que debo a proveedores?

Sí. Podés cargar deudas, vencimientos y pagos parciales o directos; los pagos asociados a caja se reflejan como egresos.

¿Cómo funciona la prueba de 7 días?

Te entregamos una licencia temporal para una PC con todas las funciones habilitadas. Después decidís si querés continuar.

¿Me ayudan a instalarlo?

Sí. Podés solicitar instalación, configuración inicial, impresora térmica y acompañamiento para tu primera puesta en marcha.
